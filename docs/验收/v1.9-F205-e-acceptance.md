## 已知限制

1. **首次部署 cache 为空**：需手动调 `POST /api/admin/refresh-pool-cache` 一次；下次周一 cron 前不自动刷新。
2. **admin endpoint 无鉴权**：内部服务可接受，后续可加 API key。
3. **universe 新增 ticker 延迟**：每月 1 日 universe 刷新后，新 ticker 需等到下个周一 cache 更新后才进入 RS 层。
