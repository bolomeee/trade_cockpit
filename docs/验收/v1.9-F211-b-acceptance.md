## 功能描述

在 DecisionPanelWidget 底部新增 "AI Contradictions" 区块：
- 默认收起（lazy trigger），点击 "Generate AI Contradictions" 按钮触发
- 复用 F208-c 的 `callAiTask` 通用 endpoint，调 `POST /api/ai/contradiction_detector`
- Input 13 字段：8 个来自 CockpitDecisionData，5 个（trendScore / rsPercentile / readySignal / regime / regimeScore）从 setupMonitor + regime react-query cache 读取
- 6 态状态机：closed / loading / error 502 / error 409 / success-with-contradictions / success-empty
- Severity badge 颜色：HIGH=--color-error，MEDIUM=--color-log-warn（--color-signal-warning 缺失，已换），LOW=--color-text-muted
