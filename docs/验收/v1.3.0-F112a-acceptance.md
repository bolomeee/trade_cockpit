## 检查结果

|------|------|------|
| 业务逻辑 B1–B6 | 6/6 | 0 |
| 边缘 E1–E3 | 3/3 | 0 |

核心确认项：
- `publishedAt` ISO-8601 带 `Z`
- `symbols` 去交易所前缀、去重、无空格
- `limit` 超出 [1,50] → 422
- FMP 上游失败 → 502 `EXTERNAL_API_ERROR`
- 默认 `limit=20`、空 list → `{"data": []}`
