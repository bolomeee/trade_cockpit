## Feature 概要

**F219 — MACD Divergence 早衰报警（最小化方案）**

在已有的 trend_score / vol_zscore / weekly_stage 体系下，仅保留 MACD divergence 一个用途：
- **bearish divergence**（价 20 日新高 + MACD 不新高）→ PositionListWidget ticker 后 ⚠️，作持仓早衰报警
- **bullish divergence**（价 20 日新低 + MACD 不新低）→ SetupMonitorWidget CAPITULATION 行 MACD+ chip，作入场辅助证据

**拆分**：F219-a（后端：indicator + DB + endpoint schema）+ F219-b（前端：widget 渲染）
